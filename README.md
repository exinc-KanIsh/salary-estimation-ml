# salary-estimation-ml
A machine learning project that predicts salary based on relevant features using data preprocessing, feature engineering, and regression models. The project includes data cleaning, model training, evaluation, and performance optimization to improve prediction accuracy.
#  Salary Prediction ML

##  Overview
A machine learning project that predicts salary based on input features using regression models. The project follows an end-to-end ML pipeline from data preprocessing to model evaluation.

---

##  Tech Stack
Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn

---

##  Workflow
- Data Cleaning & Preprocessing  
- Exploratory Data Analysis (EDA)  
- Feature Selection  
- Model Training (Regression)  
- Evaluation & Prediction
